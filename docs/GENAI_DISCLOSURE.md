# Generative AI Disclosure

This document records current and planned disclosure for generative AI assistance used while preparing this repository and grant application materials.

## Summary

Generative AI assistance was used for:

- drafting documentation structure;
- reviewing grant positioning;
- generating initial boilerplate code and tests;
- improving clarity of README and reviewer-facing docs;
- identifying missing grant-readiness artifacts.

The human author remains responsible for:

- project direction;
- technical review;
- factual accuracy;
- licensing choices;
- final code review;
- final grant submission;
- community communication.

## Current known use

| Date | Tool/model | Purpose | Human review status |
|---|---|---|---|
| 2026-06-03 | ChatGPT / GPT-5.5 Thinking | Project positioning, README improvements, runnable CLI skeleton, docs and tests | Human review required before final submission |

## Disclosure principle

The project should disclose AI assistance clearly and avoid presenting AI-generated material as unreviewed human work.

## Prompt provenance note

Before final submission, maintainers should preserve a compact prompt provenance log with:

- model or tool name;
- date and approximate time;
- purpose of the interaction;
- prompts or prompt summaries;
- whether output was used directly, edited, or discarded.

## Final submission wording draft

```text
Generative AI tools were used to assist with documentation drafting, grant-readiness review, and initial boilerplate code/tests. All project claims, code, budget, licensing, and final application text are reviewed and accepted by the human applicant, who remains responsible for the submission.
```
